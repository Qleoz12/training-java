Features: Internationalisierung

Für die Internationalisierung der Anwendung gibt es eine Reihe unterschiedlicher Ansatzpunkte:

  1. Konfiguration von message-bundle und resource-bundle in der faces-config.xml
  2. Konfiguration von unterstützten Lokalisierungen und Standard-Locale in der faces-config.xml
  3. Definition der Locale in <f:view locale="...">
  4. Definitions eines ValidationMessages.properties- Bundles für die Internationalisierung der Bean Validation Messages.
