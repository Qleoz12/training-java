Implizite Objekte in der EL

Folgende Objekte sind im Gültigkeitsbereich der EL-Auswertung standardmäßig verfügbar:

Objekt Funktion
requestScope Referenz auf die Request-Map des External-Contexts.
viewScope Referenz auf die View-Map des View-Roots .
sessionScope Referenz auf die Session-Map des External-Contexts.
applicationScope Referenz auf die Application-Map des External-Contexts.
view Referenz auf den View-Root.
param Referenz auf die Request-Parameter-Map des External-Contexts.
paramValues Referenz auf die Request-Parameter-Values-Map des External-Contexts.
header Referenz auf die Request-Header-Map des External-Contexts.
headerValues Referenz auf die Request-Header-Values-Map des External-Contexts.
facesContext Referenz auf den Faces-Context
initParam Referenz auf die Init-Parameter-Map des External-Contexts und damit auf Kontextparameter der Webapplikation.
cookie Referenz auf die Request-Cookie-Map des External-Contexts.
